# sirPizza

Simple Pizza CRUD app created with Node.js, Express, MongoDb, Mongoose, Bootstrap, EJS.


## Installing

1. Execute on your terminal: `npm install`;
2. Run your local mongoDB `sudo service mongod start`; 
3. Execute: `npm start`;
3. Open [localhost:4000](http://localhost:4000/) on your browser;

## Deployment
A working version is available on heroku by clicking [here](https://pizza-dci.herokuapp.com/)
